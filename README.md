# pdp8-tape2bin
Converts a PDP8 bin/rim tape-file into a plain binary file that is suitable to be loaded into the ram/rom/eeprom of a hardware emulator like Logisim or Digital.

```./tape2bin.sh <test.bin >test.eep```
